```typescript

const applyDoomTheme = (config: DoomTheme): void => {
  document.documentElement.style.setProperty('--primary', config.colors.primary);
  document.documentElement.style.setProperty('--secondary', config.colors.secondary);
  document.documentElement.style.setProperty('--bg', config.colors.background);
};

// Initialize the doom aesthetic
applyDoomTheme({
  colors: {
    primary: '#ff3860',
    secondary: '#00ff99',
    background: '#0a0a0a'
  }
});`}
```

## Component Properties Reference

### Alert Props

- `type`: `"info" | "success" | "warning" | "error"`
- `title?`: Optional title string
- `dismissible?`: Boolean for dismiss functionality (default: false)

### BadgeGroup Props

- `title?`: Optional group title
- `badges`: Array of strings
- `variant?`: `"default" | "success" | "error" | "warning" | "info"`

### CodeBlock Props

- `language?`: Programming language for syntax highlighting (default: "bash")
- `filename?`: Optional filename display
- `copyable?`: Enable copy button (default: true)
- `showLineNumbers?`: Show line numbers (default: true)

### ImageComparison Props

- `title?`: Optional comparison title
- `beforeSrc`: Path to before image
- `afterSrc`: Path to after image
- `beforeAlt`: Alt text for before image
- `afterAlt`: Alt text for after image

### ProgressBar Props

- `value`: Current progress value
- `max?`: Maximum value (default: 100)
- `label?`: Progress label
- `color?`: `"red" | "green" | "blue" | "yellow"` (default: "green")
- `showPercentage?`: Show percentage (default: true)
- `animated?`: Enable animations (default: true)

### Quote Props

- `author?`: Quote author
- `source?`: Quote source/context
- `type?`: `"default" | "warning" | "success" | "error"`

### StatsGrid Props

- `title?`: Grid title
- `columns?`: `1 | 2 | 3 | 4` (default: 3)
- `stats`: Array of stat objects with title, value, subtitle?, color?, trend?

### Timeline Props

- `title?`: Timeline title
- `items`: Array with title, date, description?, status? ("completed" | "current" | "upcoming")

### Tabs Props

- `defaultTab?`: ID of default active tab
- `items`: Array with id, title, and content (React.ReactNode)
